# Vendas - Semana Dev Superior 3

### Project ###
Application for viewing Sales Dashboards. Through pagination it is possible to view the sales in chronological order and table format, with the following information: 
* Date of sale
* Salesperson
* Clients visited
* Closed deals
* Value

### Technologies ###
#### Back end ####
* Java - Spring Boot

#### Front end ####
* TypeScript 
* ReactJS
